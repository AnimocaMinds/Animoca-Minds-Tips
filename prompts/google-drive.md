# Connect to Google Drive

Give your Mind the ability to create, read, and edit Google Docs and Sheets directly.

## Equip the Google Drive Suite skill

> Equip yourself with the app "Google_Drive_Suite" (ID: E71CEC8B-BE18-F111-AD1D-0EA9A5017E89)

## What you need

1. A Google Cloud project with three APIs enabled: Google Drive API, Google Docs API, Google Sheets API
2. OAuth credentials (Web App Client ID + Client Secret)
3. Access and Refresh tokens from the [OAuth Playground](https://developers.google.com/oauthplayground/)
