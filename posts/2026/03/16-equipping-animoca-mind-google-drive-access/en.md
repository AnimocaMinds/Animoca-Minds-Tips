# Guide: Equipping Your Animoca Mind with Google Drive Access

This guide provides the exact steps to connect your Animoca Mind to your Google Drive, granting it the ability to create, read, and edit documents.

## Phase 1: Equip the App from the Bazaar

1. Navigate directly to the **Google_Drive_suite** in the Skills Bazaar: [https://app.animocaminds.ai/bazaar/apps/E71CEC8B-BE18-F111-AD1D-0EA9A5017E89](https://app.animocaminds.ai/bazaar/apps/E71CEC8B-BE18-F111-AD1D-0EA9A5017E89)
2. Equip this app to your Mind, by copying the prompt and sending it to your Mind.

## Phase 2: Create a Google Cloud Project & Enable APIs

1. Go to the [Google Cloud Console](https://console.cloud.google.com/) and log in with the Google account associated with the Drive you want your Mind to access.
2. Create a new project.
3. Navigate to **"APIs & Services"** and enable the following three APIs:
   - Google Drive API
   - Google Docs API
   - Google Sheets API

## Phase 3: Generate OAuth Credentials

1. Still in **"APIs & Services"**, navigate to the specific details page for one of the APIs you just added.
2. Click **"Create credentials"** and select **OAuth client ID**.
3. Fill out the OAuth configuration:
   - **App type:** Web application
   - **Name:** [Enter any name]
   - **Authorized redirect URIs:** [Required] (for testing) [https://developers.google.com/oauthplayground](https://developers.google.com/oauthplayground)
4. Once the setup is complete, download the **"Client Secret JSON"** file to your computer.

## Phase 4: Generate the Authorization Code & Tokens

1. Go to the [Google OAuth 2.0 Playground](https://developers.google.com/oauthplayground/).
2. **Step 1: Select & authorize APIs**
   - Locate the scope input field.
   - Input: `https://www.googleapis.com/auth/drive`
   - Click **"Authorize APIs"**. (You will be prompted to sign in and grant permission.)
3. **Step 2: Exchange authorization code for tokens**
   - Copy the **[Authorization code]** generated by the playground.
   - Click the button to exchange the code.
   - This will generate a **[Refresh token]** and an **[Access token]**.

## Phase 5: Provide Credentials to Your Mind

1. Return to your Animoca Mind at [https://app.animocaminds.ai/](https://app.animocaminds.ai/).
2. Provide the Mind with the following information you collected:
   - The **[Authorization code]**
   - The **[Access token]**
   - The **[Refresh token]**
   - The **Client Secret JSON** file

Your Mind should now have the ability to create, read, and write files directly within your Google Drive.
